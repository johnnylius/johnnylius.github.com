实现默认方式支持HTTPS

首先恭喜你，无论你使用NSURLConnection，NSURLSession还是AFNetworking，默认已经支持了HTTPS，不用写任何代码。当然这也需要一些小的前提，服务端证书是有效证书，并且包含所请求的域名。但如果是自签名方式，则需要自己写代码实现。
